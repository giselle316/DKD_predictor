# Prediction Model for 1-Year Renal Composite Outcome in DKD Patients at CKD Stage 3-4 Using Machine Learning

## Environment
- Python 3.13.2
- Jupyter Notebook

## How to Run
1. Launch Jupyter Notebook
2. Open the `.ipynb` files in your browser in numerical order  (they are numbered according to the workflow) and run each file sequentially to reproduce the results reported in the paper.
