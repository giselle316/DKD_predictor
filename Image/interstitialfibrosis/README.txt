# Deep Learning-Based Renal Pathology Image Classification Model

## Environment
- Python 3.11.13
- Jupyter Notebook

## File Description
- `Fibrosislabel.xlsx` : Annotation file for training images
- `fibrosismodels.ipynb` : Notebook for training the model
- `F.pth` : The final trained model
- `Ftest.xlsx` : Annotation file for testing images
- `test.ipynb` : Notebook for testing the model

## How to Run
Open Jupyter Notebook and run the `.ipynb` files as needed to reproduce training and testing results.
