# Deep Learning-Based Renal Pathology Image Classification Model

## Environment
- Python 3.11.13
- Jupyter Notebook

## File Description
- `C.pth` : The final trained model
- `Crescentlabel.xlsx` : Image annotation file
- `crescent_model.ipynb` : Notebook for training the model
- `crescent_heatmap.ipynb` : Notebook for generating heatmaps

## How to Run
Open Jupyter Notebook and run the code in each `.ipynb` file.
